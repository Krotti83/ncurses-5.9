# Unofficial repository for ncurses-5.9

`ncurses-5.9` is required in various proprietary software like AMD's (former Xilinx) Vivado/Vitis
to run these software on GNU/Linux operating systems. Unfortunately mostly of the GNU/Linux
distributions no longer include `ncurses-5.9` in the default packages. So therefore it is
required to build `ncurses-5.9` by yourself.
